# oh-my-zsh-config

## Symbolic Link
### .zshrc
> ln -s ~/workspace/eclectic-bits/oh-my-zsh-config/zshrc ~/.zshrc

### oxide.zsh-theme
> ln -s ~/workspace/eclectic-bits/oh-my-zsh-config/themes/oxide.zsh-theme ~/.oh-my-zsh/custom/themes/oxide.zsh-theme

## reload zsh
> source ~/.zshrc
